Descrição: Uma instituição de ensino busca criar uma plataforma de aprendizado online que
se adapte ao ritmo e estilo de cada aluno. O sistema deve oferecer trilhas de aprendizado
personalizadas, avaliações dinâmicas e feedback instantâneo, utilizando IA para identificar
lacunas de conhecimento e sugerir conteúdos.
Desafios Arquiteturais:
• Personalização e IA: Motor de recomendação e adaptação de conteúdo.
• Escalabilidade: Suportar milhares de alunos simultaneamente.
• Integração: Conexão com sistemas de gestão acadêmica (LMS) e repositórios de
conteúdo.
• Usabilidade: Interface intuitiva e responsiva para diferentes dispositivos.
Requisitos Não Funcionais Chave: Escalabilidade, Usabilidade, Manutenibilidade,
Performance.
